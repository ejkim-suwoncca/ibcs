The Ghost in the Machine

How does a "silent box of silicon" wake up?

Sequence: Power → Storage → CPU → RAM

Step 1: Secondary Storage

This is the "Long-Term Memory." It holds the OS (C-Drive).

Characteristics: Non-volatile (Data stays when power is off).

Step 2: The CPU

The Central Processing Unit "fetches" instructions from storage.

Key Parts: CU (Control Unit) & ALU (Math).

Step 3: Primary Memory (RAM)

The "Short-Term Workspace" where active programs live.

Characteristics: Volatile (Data disappears without power).

The Von Neumann Architecture

Data and Instructions live together in the same memory.

Hodder Textbook p. 8-9

  • MAR: Memory Address Register (The Location)
  • MDR: Memory Data Register (The Content)

Why is one PC faster?

  • Clock Speed: How many cycles per second (GHz).
  • Cores: Multiple "brains" working at once.
  • Cache: Ultra-fast memory inside the CPU.
