// Slide 2 · Guiding Question

How can binary represent
numbers, text, and images?

This lesson follows one big pattern: take something meaningful, turn it into a numerical code, then store that code using only 0s and 1s.

Numbers
Place value in base 2
Each bit position has a value such as 1, 2, 4, 8, and 16.
Text
Characters become codes
Letters and symbols are assigned numbers, then stored in binary.
Images
Pixels become data
Each pixel stores a colour value, and those values are written in binary.

Guiding idea: binary is not just for counting. It is a universal way to encode different kinds of information.

// Unicode · The World's Character Set

ASCII → Unicode

ASCII only has 128 characters — enough for English, but not the world's languages. Unicode solves this.

ASCII — Limited
128 characters (7 bits)
English letters, numbers, basic punctuation only.
Cannot represent Korean, Chinese, Arabic, emoji…
Unicode — Universal
Over 140,000 characters
Covers all writing systems on Earth — Latin, Hangul, Chinese, Arabic, Devanagari, emoji, and more.

Backward compatible: the first 128 codes are identical to ASCII.

한국어 (Korean / Hangul) is part of Unicode. The Hangul block covers U+AC00 → U+D7A3 — 11,172 syllable blocks.

// Colour Depth

Colour Depth — Bits per Pixel

More bits per pixel = more possible colours = better image = larger file.

Colour DepthBits/PixelMax ColoursFormulaUse Case
1-bit 1 2 2¹ Black & White only
8-bit 8 256 2⁸ Greyscale / GIF
24-bit 24 16.7M 2²⁴ (8R+8G+8B) True colour JPEG/PNG
32-bit 32 16.7M + transparency 2³² (RGB + Alpha) Games, UI, PNG

More bits per pixel → more colours → larger file

24-bit = 8 bits each for Red, Green, Blue. 256 × 256 × 256 = 16.7M combinations.

// Sound · Analog vs Digital

Sound is Analog

KEY TERM ANALOG
Analog = a signal that is continuous — it has infinite possible values at every moment in time.

Sound in the real world is analog: air pressure changes smoothly and continuously.

Computers are digital — they can only store discrete (stepped) values. So analog sound must be converted to digital.
KEY TERM AMPLITUDE
Amplitude = the height of the sound wave — how loud the sound is.

When a computer samples sound, it records the amplitude at each point as a number.
ANALOG WAVE (continuous)
DIGITAL (sampled — discrete steps)
// Sound Quality

Sample Rate & Bit Depth

Two numbers control sound quality. Together they determine how accurately digital audio recreates the original analog wave.

📊 Sample Rate
Hz / kHz
Samples taken per second.

44,100 Hz = CD quality (standard)
48,000 Hz = professional video
96,000 Hz = studio recording

Higher → smoother, more accurate sound
📏 Bit Depth
bits / sample
Bits used to store the amplitude of each sample.

8-bit = 256 amplitude levels
16-bit = 65,536 levels (CD)
24-bit = 16.7M levels (studio)

Higher → more precise amplitude values

// Video Representation

Video = Frames + More

A video file is not just moving pictures — it's a combination of several data types bundled together.

🎞️
Frames
Sequence of still images. Each frame = one picture. Played rapidly to create motion.
🎵
Audio
Sampled sound data. Synchronised with the frames. Stored as binary (sample rate + bit depth).
📋
Metadata KEY TERM
Data about the video — title, duration, resolution, codec, creation date, GPS location.
💬
Subtitles
Text track with timestamps. Stored as a separate data stream. Can be multiple languages.

👉 Focus: The frames make the video possible — everything else supports them. Frames are also the most data-intensive component.

// Video · Frames & Compression

Frames — Why Compression Matters

What is a Frame?
A frame is a single still image. Video is just many frames played one after another very quickly.

FPS (Frames Per Second) = how many images play per second.

24 fps = cinema  ·  30 fps = standard video  ·  60 fps = smooth gaming
⚠️ The Problem — Uncompressed Video is Huge
1 frame at 1080p (24-bit) ≈ 6 MB
At 30 fps: 6 MB × 30 = 180 MB per second
1 minute of video = ~10 GB uncompressed

This would be impossible to stream or store efficiently.

👉 Solution: Compression. Frames are compressed using codecs (H.264, H.265, AV1) to make video practical.

// Metadata

Metadata — Data About Data

Metadata = information that describes a file rather than being the content itself. KEY TERM

What Metadata Tells You
Metadata answers: What is this file? When was it made? Where? How?

It doesn't contain the actual content — it describes the context around the content.

Every file you create — photo, video, document — contains metadata automatically, often without you knowing.

EXAMPLES BY FILE TYPE
📸 IMAGE (JPEG/PNG)
Camera model, GPS location, date taken, shutter speed, ISO, resolution
🎬 VIDEO (MP4)
Duration, resolution, frame rate, codec, audio channels, creation date, title
🎵 AUDIO (MP3)
Artist, album, track number, duration, bit rate, genre
📄 DOCUMENT (PDF/DOCX)
Author, date created/modified, word count, page count, software used
// Integers · Signed vs Unsigned

Signed vs. Unsigned

The same 8 bits can represent two completely different ranges — depending on whether the value is signed or unsigned.

Unsigned — Positive only
All 8 bits = magnitude.

Range: 0 to 255

Used for: pixel colour values, ASCII codes, memory addresses, file sizes
Signed — Positive & Negative
The MSB (leftmost bit) = sign bit.

0 = positive  ·  1 = negative

Range: −128 to +127

Used for: temperatures, audio amplitude, coordinates, game scores
Bit 7 (MSB)Bit 6Bit 5Bit 4 Bit 3Bit 2Bit 1Bit 0 Value
Unsigned 1000 0010 130
Signed 1000 0010 −126

Same bits, different interpretation. The system decides whether the MSB is a sign bit or part of the value.

Signed: MSB = sign bit → only 7 bits for magnitude. Range: −128 to +127 (2⁷ = 128).

// Vocabulary · Key Terms

Terms to Know

ASCII
128-character encoding standard. Maps characters to numbers 0–127.
Unicode
Universal encoding covering 140,000+ characters. Includes Korean (한글), all world languages, emoji.
Pixel
Smallest unit of a digital image. Each has a colour value stored as binary.
Image Resolution
Total number of pixels (width × height). More pixels = more detail.
Analog
A continuous signal with infinite values. Sound in the real world is analog before it is digitised.
Amplitude
Height of a sound wave = loudness. Each audio sample records the amplitude at that moment.
Frame
One still image in a video. Rapid playback of frames creates motion (fps).
Metadata
Data that describes a file — not the content itself. E.g. date, author, resolution, GPS.
Signed / Unsigned
Unsigned: 0–255. Signed: uses MSB as sign bit, −128 to +127.
