// B2.3.3 · Iteration

Loops Repeat Instructions

Iteration means repeating a block of code. Use a loop when the same action should happen more than once.

counted loop
Use when the number of repetitions is known.

for i in range(5)
conditional loop
Use when repetition depends on a condition.

while password != "ib2027"
loop body
The indented code that repeats each time the loop runs.

Common error: Forgetting to indent the loop body in Python means the code is not part of the loop.

// B2.3.3 · Counted Loops

Using range() in Python

A counted loop repeats a known number of times. In Python, range() creates the numbers the loop will use.

Common range patterns
for i in range(5): print(i) # 0, 1, 2, 3, 4 for i in range(1, 6): print(i) # 1, 2, 3, 4, 5 for i in range(2, 11, 2): print(i) # 2, 4, 6, 8, 10
Common errors
range(5) starts at 0, not 1.

The stop value is not included, so range(1, 6) stops after 5.

If you need exactly five repeats, use range(5). If you need the numbers 1 to 5, use range(1, 6).

Practice: Write a loop that prints the multiples of 3 from 3 to 30.

// B2.3.3 · Counted Loop Examples

Use a Counted Loop When the Amount Is Known

Add five scores
total = 0 for i in range(5): score = int(input("Score: ")) total = total + score average = total / 5 print("Average:", average)
Build a simple output pattern
for row in range(1, 6): print("*" * row) # output: # * # ** # *** # **** # *****

Common error: Updating total outside the loop only adds one score. The update line must be indented inside the loop body.

// B2.3.3 · Conditional Loop Examples

Use a While Loop When the Amount Is Unknown

A conditional loop keeps running while its condition is True. It is useful when the program does not know how many attempts will be needed.

Input validation
age = int(input("Age: ")) while age < 0: print("Age cannot be negative.") age = int(input("Age: ")) print("Age saved:", age)
Sentinel value
total = 0 score = int(input("Score (-1 to stop): ")) while score != -1: total = total + score score = int(input("Score (-1 to stop): ")) print("Total:", total)

Common error: A while loop must change something used in the condition. If score is never updated, the loop may never stop.

// B2.3.3 · Selection Within Iteration

Use if Inside a Loop to Filter

Loops often repeat the same check for every value in a collection or for every user input.

Count passing scores
scores = [45, 72, 88, 39, 61] passes = 0 for score in scores: if score >= 50: passes = passes + 1 print("Passes:", passes)
Find the largest value
marks = [64, 91, 78, 85] highest = marks[0] for mark in marks: if mark > highest: highest = mark print("Highest:", highest)

Practice: Change the first example so it counts scores below 50 instead of scores 50 or higher.

// B2.3.3 · Iteration (Loops)

Counted Loops vs Conditional Loops

Use a counted loop (for) when you know in advance how many times to repeat. Use a conditional loop (while) when you repeat until a condition changes.

Counted loop — for (known repetitions)
# Python total = 0 for i in range(5): # runs exactly 5 times score = int(input("Score: ")) total += score print("Total:", total)
// Java int total = 0; for (int i = 0; i < 5; i++) { int score = scanner.nextInt(); total += score; } System.out.println(total);
Conditional loop — while (unknown repetitions)
# Python password = "" while password != "ib2027": password = input("Password: ") print("Access granted")
// Java String password = ""; while (!password.equals("ib2027")) { password = scanner.next(); } System.out.println("Access granted");

Avoid infinite loops: Every while loop needs a variable that changes inside the loop so the condition can eventually become False. If nothing changes, the loop runs forever.

Conditional statements within loops: You can use if inside a loop to filter or branch — for example, counting only the positive numbers in a list.

// B2.3.4 · Functions & Modularization

Breaking Programs Into Focused, Reusable Parts

Python
def letter_grade(score): # score is a local parameter if score >= 80: return "A" elif score >= 60: return "B" else: return "C" grade = letter_grade(74) print(grade) # B
Java
static String letterGrade(int score) { if (score >= 80) return "A"; else if (score >= 60) return "B"; else return "C"; } String grade = letterGrade(74); System.out.println(grade); // B
ConceptMeaningBenefit
ParameterInput value supplied to the functionMakes the function flexible — works for any score
Return valueOutput sent back to the callerThe result can be stored and reused anywhere
Local scopeVariables created inside the functionPrevents accidental interference with other parts
ModularizationSplitting a program into named, focused functionsEasier to read, test, debug and maintain

Common error: Printing a value inside a function is not the same as returning it. Use return when the caller needs to store or reuse the result.
