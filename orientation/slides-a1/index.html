Phase 1: The Spark

What happens when you press Power?

  • 1. BIOS (ROM) wakes up.
  • 2. CPU looks at the Hard Drive (C-Drive).
  • 3. The "Fetch" begins.

Phase 2: The Fetch

The CPU fetches the Operating System from Secondary Storage.

Note: Storage is NON-VOLATILE (data stays when power is off).

Phase 3: The Load (RAM)

The CPU moves active programs into **RAM**. RAM is fast, but it is VOLATILE. If power cuts, data vanishes!

The CPU Components

  • Control Unit (CU): The Manager. Fetch, Decode, Execute.
  • ALU: The Calculator. Math and Logic.

Why is some tech faster?

Clock Speed: How many cycles per second? (measured in GHz).

Cores: How many "brains" working at once?

AI Architect Tip: NVIDIA GPUs have thousands of cores to process AI math simultaneously.
